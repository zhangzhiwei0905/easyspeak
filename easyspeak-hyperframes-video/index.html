Quiz + review

复习和测验，跟着记忆节奏走。

错题回顾、主题测验、闪电测验和间隔复习，让用户知道今天该巩固什么。

System architecture

一个小程序，一个后端，一条内容流水线。

前端保持原生小程序体验；FastAPI 负责认证、内容、进度和测验；AI 参与生成更自然的干扰项。

WeChat Mini Program
首页、词库、复习、测验、个人中心，贴近日常使用路径。
FastAPI Backend
JWT 登录、每日内容、学习进度、复习调度与统计。
AI Quiz Support
根据真实内容生成干扰项，让练习更接近语言使用。
